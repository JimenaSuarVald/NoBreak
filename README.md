# NoBreak
Disfruta de tu musica sin anuncios constantes e interrupciones molestas. 

Este software fué diseñado por 
  - Jimena Suárez-Valdés
  - Diego Enriquez Carrera
  - Sara Román Plaza 
  - Angel Yang

La filosofía de este software es tener un acercamiento a la musica de manera accesible y sencilla, sin empresas predatorias que utilicen sus monopolios
multinacionales para determinar como la insustria musical ha de manejarse.

Hecho por 6 estudiantes sin animo de lucro para el disfrute del usuario.
